# Mô-đun 16: Tích Hợp Steamworks Qua GodotSteam Plugin

## 1. Tính Năng Steam Integation
- **DRM Launch Enforcement**: Khởi động game qua Steam Client (`Steam.restartAppIfNecessary`).
- **Steam Achievements**: Mở khóa 50+ thành tựu đồng bộ trực tiếp lên profile Steam của người chơi.
- **Steam Cloud Save**: Tự động lưu save file lên mây Steam Cloud.

## 2. File Cấu Hình Local Dev
File `steam_appid.txt` chứa Steam App ID thử nghiệm được đặt tại thư mục gốc của Godot project.
