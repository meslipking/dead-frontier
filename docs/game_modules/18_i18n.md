# Mô-đun 18: Đa Ngôn Ngữ i18n (Tiếng Việt & English)

## 1. Cơ Chế Godot TranslationServer
Hệ thống sử dụng các file CSV chuẩn (`locale/vi.csv`, `locale/en.csv`) nạp vào `TranslationServer` của Godot Engine.

## 2. Hàm Chuyển Đổi Ngôn Ngữ
Người chơi có thể chuyển đổi qua lại giữa Tiếng Việt và Tiếng Anh tức thì tại Trung Tâm Chỉ Huy mà không cần khởi động lại game.
