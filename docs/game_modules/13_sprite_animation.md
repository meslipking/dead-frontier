# Mô-đun 13: Đồ Họa Pixel Art & Sprite Animation Engine

## 1. Sprite Sheet Standards
- Nhân vật Sống sót: Grid 32x32px.
- Quái vật & Mecha: Grid 64x64px.
- Các Animation Loop chuẩn: Idle (4 frames), Walk (6 frames), Attack (4 frames), Hit (2 frames), Death (4 frames).

## 2. Godot AnimatedSprite2D Setup
Sử dụng tài nguyên `SpriteFrames` tích hợp sẵn trong Godot Engine để chạy hoạt ảnh frame-based mượt mà trên phần cứng GPU 2D.
