# Mô-đun 11: 7 Vùng Hoang Thám Hiểm (Wasteland Zones)

## Danh Sách 7 Vùng Hoang
1. **Ngoại ô hoang tàn (Difficulty ⭐)**: Zombie thông thường, nguyên liệu phế liệu sắt.
2. **Cống ngầm độc (Difficulty ⭐⭐)**: Quái vật hệ Độc, nguyên liệu dung dịch độc.
3. **Nhà máy bỏ hoang (Difficulty ⭐⭐⭐)**: Robot nổi loạn & zombie thép, linh kiện Mecha.
4. **Cảng đóng băng (Difficulty ⭐⭐⭐⭐)**: Sinh vật hệ Băng, pha lê băng giá.
5. **Hố bom hạt nhân (Difficulty ⭐⭐⭐⭐⭐)**: Biến dị phóng xạ, mảnh hạt nhân.
6. **Tổ zombie (Difficulty ⭐⭐⭐⭐⭐⭐)**: Hang ổ trùm zombie, DNA đỉnh cao.
7. **Pháo đài trên mây (Difficulty ⭐⭐⭐⭐⭐⭐)**: Hệ thống AI quân sự, bản thiết kế Mecha huyền thoại.
