# Mô-đun 17: Xuất Bản Mobile (Android APK & iOS IPA)

## 1. Tối Ưu Cấu Hình Export Preset
- **Android**: Target SDK API 34, Android 7.0 (API 24) trở lên, cấp quyền INTERNET.
- **iOS**: Sign bản build bằng Apple Developer Account (Development / Distribution).

## 2. Quy Chuẩn Giao Diện Cảm Ứng (Touch Target Standards)
Tất cả các nút tương tác (Buttons, Cards) có kích thước tối thiểu **44x44dp** đảm bảo thao tác ngón tay chính xác trên mọi màn hình điện thoại di động.
