# Mô-đun 02: Hệ Thống Giao Diện Design System & Bảng Màu SNES

## 1. Phong cách Đồ họa
- **Art Style**: SNES-era 16-bit Pixel Art (Sprite kích thước 32x32 và 64x64 pixels).
- **Giao diện UI**: Dark Post-Apocalyptic Glassmorphic UI (Bề mặt tối trong suốt, viền ánh kim sắc nét).

## 2. Bảng Màu Chuẩn (Color Palette)
- **Background Tối**: `#0a0d12` (Màn hình chính), `#151922` (Panel thẻ)
- **Tông Màu Chủ Đạo**: `#e85d3a` (Đỏ cam tận thế - Primary Action)
- **Tông Màu Phụ**: `#4ecdc4` (Xanh độc hóa - Secondary Highlight)
- **Cảnh Báo & Nhấn**: `#ffd93d` (Vàng cảnh báo - Accent)

### Bảng Màu Cấp Độ Hiếm (Rarity Color Scheme)
| Cấp Hiếm | Tên Tiếng Việt | Mã Màu Hex | Color RGB (Godot) |
|---|---|---|---|
| Common | Thường | `#9ca3af` | Color(0.61, 0.64, 0.69) |
| Uncommon | Khá | `#4ade80` | Color(0.29, 0.85, 0.50) |
| Rare | Hiếm | `#60a5fa` | Color(0.38, 0.65, 0.98) |
| Epic | Sử thi | `#a78bfa` | Color(0.65, 0.55, 0.98) |
| Legendary | Huyền thoại | `#fbbf24` | Color(0.98, 0.75, 0.15) |
| Mythic | Thần thoại | `#f43f5e` | Color(0.96, 0.25, 0.37) |

## 3. Quy chuẩn Font chữ & Tiếng Việt
- Sử dụng Font Pixel chuyên dụng hỗ trợ đẩy đủ bảng mã Tiếng Việt Unicode có dấu.
- **Font Sizes**: Title (20px), Header (16px), Body (14px), Subtext (11px), Micro (10px).
